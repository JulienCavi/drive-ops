Grille d'édition

Mode rayon guidé

Cliquer sur la grille pour placer les meubles. Utiliser le mode rayon guidé pour aller plus vite : MT00, MT01, MT02… s'enchaînent automatiquement.

Aucun rayon en cours. Cliquer sur une case pour éditer manuellement.

Instructions

Astuce : Il est possible de corriger une case en dehors du mode rayon guidé. Cliquer sur une case → modifie le code, ou lauisser vide pour effacer.

Sauter : En mode guidé, utiliser le bouton "Sauter" pour passer au meuble suivant sans placer le courant (utile si un meuble n'existe pas).
